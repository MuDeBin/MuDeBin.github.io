5.解决WPS字体缺失方法

找到提示缺失的字体 打开/usr/share/fonts文件夹。
在里面新建文件夹 当前目录右键打开终端 ：
$ sudo mkdir wps-fonts
或者 可以直接复制进来就可以：
$ sudo cp （你字体保存路径） /usr/share/fonts/wps-fonts
然后执行：
$ sudo mkfontscale
$ sudo mkfontdir // 生成字体索引
最后更新缓存：
$ sudo fc-cache
