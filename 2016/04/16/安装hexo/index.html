hexo主题的安装和简单配置

  • 知识管理

hexo创建和部署到github上

后记

默认是自己的github上在你的账户下存在一个仓库 格式如
ABC(github的名字)/ABC.github.io ABC.github.io 格式的仓库默认只允许建立一个

安装hexo

$ npm install -g hexo

hexo文件夹的创建

在你想放置hexo主题的磁盘目录下建立hexo文件夹 （D: \HEXO）
在 该文件夹目录下右键 ---> git bash here
$ hexo init

安装依赖包

$ npm install

本地部署

$ hexo g     g = generate  (生成)
$ hexo s     s = server   (启动本地服务)

在本地预览hexo主题的博客

打开浏览器，在浏览器地址栏输入：
                localhost:4000  & http://127.0.0.1:4000

修改配置 _config.yml 文件位置：hexo文件夹下

找到deploy：
  deploy:
    type: git
    repository: https://github.com/youname/youname.github.io.git
    branch: master

安装一个推送插件

$ npm install hexo-deployer-git --save

部署到github

$ hexo d     d = deploy
