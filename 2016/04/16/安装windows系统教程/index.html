windows系统安装

windows学习总结win 7/10
  • 学习总结

windows 7/10 系统USB安装教程，抱歉没图，以后改进

题记

现在好多人都对电脑很熟悉了，但是时间用久了或者出现系统故障了，就会面临先要重装系统的想法

排除自己是那种不想学点新东西的人

装系统还的自己亲自动手？？你 out 了，谁的电脑不装个360、电脑管家神的，第一可以杀毒，清垃圾啥的。其实我们的电脑小白用户完全不用装这些，占用大量资源不说，拖慢系统，造成卡顿，全家桶的情况时常发生。我也不是在这说这个，跳过。哈哈，其实这些软件里都有一键重装系统。我只了解过，不怎么熟，本人从来不用。

先去下载纯系统，不要使用ghost的系统

推荐一个网站，微软的正版软件都在里面，我比较喜欢。
百度直搜 ：msdn 选择左侧菜单 操作系统 下载自己想要的系统

准备一个不小于8G的U盘

这里我也是用喜欢那个PE环境来装
网上搜U盘启动盘的制作，一堆！

准备

把下好的系统拷在刚弄好的启动盘里。

重新启动电脑，看自己的电脑品牌，不造的百度查，选择F2 / F12 / ESC等这些快速进入开机启动项选择 ，选择U盘进入PE环境 。

bios设置启动开机默认U盘，进入Bios方法（看品牌查进入bios的方式），选择 U盘为第一启动项。

其他的PE里怎么安装，在你制作启动盘的软件里就有教程，巨细。我也不在此啰嗦，

不用PE环境安装

磁盘分区是MBR

  1. 找一个不含资料的空U盘，把ISO格式的系统解压到U盘
  2. 开机选择U盘启动，直接进入安装界面。
