hexo 使用前git的简单配置

githexo
  • 学习总结

hexo创建中git命令简单操作
这些命令都在 git bash 完成

1. 装完git后需要配置自己的github仓库的地址

$ git config --global user.name  "这里填写自己github账号的名字" + 回车  
$ git config --global user.email "这里填写注册时所用的邮箱" + 回车   example : 12345678@qq.com

2. 检查电脑本地是否存在 SSH key

$ ~/.ssh   提示No such file or directory 说明你是第一次使用git   一般还没配置都没有 
$ ssh-keygen -T rsa -C "自己github的邮箱"  + 回车
            会提示你是否需要输入一个密码来加密，以保证被提交到github上的命令是自己提交的
            建议使用密码 
            在本地 C:\Users\用户名\.ssh 文件夹
               里面包含私匙  id_rsa
                      公匙  id_rsa.pub

3. 提交公匙到github

1. 登录github网站 ---> 右上角点击自己的账号名 ---> setting  ---> SSH and GPG keys  ---> 右上角 New SSH key    
 2. 用比如notepad++ 或者sublime text 工具打开 id_rsa.pub  Ctrl+A ---> Ctrl + C     ---> Ctrl +V 填入网页的key 栏 Title自己随便写
 3. Add SSH key

4. 检查是否匹配

$ ssh -T git@github.com   
  Warning: Permanently added the RSA host key for IP address '192.30.252.130' to the list of known hosts.
  Enter passphrase for key '/c/Users/RoyalBloodSky/.ssh/id_rsa':
  Hi ***! You've successfully authenticated, but GitHub does not provide shell access.
  如有上述提示，恭喜你匹配成功！

你又离成功近了一步！
