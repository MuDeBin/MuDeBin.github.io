Androidgithub学习总结

AndroidStudio关联github,让我们远程提交代码到github
—提示：所有以下操作是在安装好git并远程连接上github —

1.配置Git

a. 运行AndroidStudio,选择 File-->Setting

b. 选择Version Control --> Git

c. 在右侧path里找到git安装的路径，点击test，弹出提示，说明连接成功。

2.链接github

a. Setting-->Version Control-->github 。输入自己github的账号

b. 点击Test 提示如下则成功

3.提交

a.现在我们就可以提交自己的项目到github了，步骤：VCS-->蓝色选项

b. 自己设置了提交密匙
